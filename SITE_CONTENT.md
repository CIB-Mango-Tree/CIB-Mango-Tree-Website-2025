## Methodologies (Interactive Tabs)

**Section Title:** Methodologies

Each methodology appears as a clickable tab. Below is the content for each:

### Copy Pasta Test
> This test looks for repeated phrases‚Äî"copypasta"‚Äîthat appears across posts in your dataset. Some coordinated campaigns may consist of bot accounts reposting the same phrases to cut down on time. Our test extracts these repeated phrases, and reveals which accounts posted them, and when, in order to reveal hidden networks.

### Hashtag Test
> The Hashtag Test analyzes hashtag usage patterns across your dataset to identify coordinated behavior. It detects unusual hashtag clustering, timing patterns, and account associations that may indicate inauthentic coordination. This helps reveal networks that use hashtags strategically to amplify messages.

### More Coming Soon!
> We are continuously developing new tools to help detect coordinated inauthentic behavior. Stay tuned for additional tests and features that will help you analyze social media datasets more effectively.

## Glossary (expandable accordion)

**Section Title:** Glossary of Terms

Each term expands when clicked to show the definition:

**APIs (Application Programming Interfaces)**
> Tools that allow different software programs to communicate with each other, often used to access data from social media platforms.

**Bot**
> An automated account that performs tasks on social media, often used to artificially amplify content or spread disinformation at scale.

**Coordinated Inauthentic Behavior (CIB)**
> Organized, malicious efforts to manipulate public opinion online through fake accounts or disinformation.

**Civic Tech**
> The use of technology to improve the relationship between citizens and their government.

**Scraping**
> The automated process of extracting data from websites using custom-built programs or tools, often called "scrapers."

**Command Line Interface (CLI)**
> A text-based interface on your computer that allows you to interact with software by typing commands.

## Preparing Your Data (yellow explainer box)

**Section Title:** Preparing Your Data

**Introduction:**
> To get the most accurate results from CIB Mango Tree, ensure your social media dataset is correctly formatted.

**Accepted File Formats:**
- CSV (Comma Separated Values) ‚Äî Recommended
- JSON (JavaScript Object Notation)
- XLSX (Microsoft Excel)

**Required Columns:**
| Column | Description | Required? |
|--------|-------------|-----------|
| post_id | Unique identifier for each post | Yes |
| text | Full text content of the post | Yes |
| author_id | Unique identifier for the author | Yes |
| timestamp | When the post was created (ISO 8601) | Yes |
| platform | Social media platform (e.g., Twitter) | No |

**Data Cleaning Steps:**
1. **Format Check** ‚Äî Ensure data is in a supported format (CSV recommended)
2. **Column Mapping** ‚Äî Verify required columns are present and properly named
3. **Data Cleaning** ‚Äî Remove duplicates and handle missing values
4. **Validation** ‚Äî Run data validation before analysis

**Link:** Full guide at https://cibmangotree.org/pages/data-preparation/data-preparation.html
